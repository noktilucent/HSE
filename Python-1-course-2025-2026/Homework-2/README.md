## Тема 3. Функции и области видимости

### Задание 1
Функция `sum_distance(from_, to_)` суммирует все числа от значения `from_` до `to_` включительно.  
Если первое число больше второго - они меняются местами.

### Задание 2
Функция `trim_and_repeat(text, offset=0, repetitions=1)` обрезает строку слева на `offset` символов и повторяет её `repetitions` раз.  
По умолчанию `offset=0`, `repetitions=1`.
